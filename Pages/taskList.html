Task List 2022 Project

Project Overview

This project is a simple task list application that allows users to add, save, edit, and delete tasks. The front-end is built using HTML, CSS, and JavaScript, while the data is being stored in a temporary cache. Users can easily enter their task details using the input field and clicking the "Add Task"button. Tasks will then be displayed in a list format and can be edited or deleted by clicking the corresponding buttons. This application provides the user with a responsive design and functionality, making it easy for users to manage their tasks. A CSS template provided by TylerPottsDev was used to style the application and provided a foundation for the project while allowing me to focus on implementing the functionality.
